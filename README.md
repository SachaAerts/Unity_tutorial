# Tutorial Unity

## Description

Projet Unity réalisé dans le cadre d'un stage, servant de support pour s'initier au développement de jeux avec Unity.

Le jeu est un **survival top-down** : le joueur contrôle un personnage vu de dessus et doit survivre à des vagues d'ennemis qui se dirigent vers lui. Pour se défendre, le joueur peut tirer sur les ennemis. La partie se termine lorsque le joueur a perdu ses 3 vies.

## Gameplay

- **Vue** : Top-down (dessus)
- **Objectif** : Survivre le plus longtemps possible face aux ennemis
- **Contrôles** : Déplacement du personnage + tir sur les ennemis
- **Game Over** : Le joueur perd après avoir reçu 3 coups

## Technologie

- **Moteur** : Unity
- **Langage** : C#

## Comment jouer

1. Cloner le repository
2. Ouvrir le projet dans Unity Hub
3. Lancer la scène principale depuis l'éditeur Unity

## Objectifs du projet

Ce projet est utilisé comme support d'apprentissage pour maîtriser les concepts fondamentaux de Unity, notamment :

- Création et gestion des préfabs
- Système de détection de collisions
- Gestion des ennemis et de leur comportement (IA basique)
- Système de tir (raycast)
- Gestion de l'état du jeu (vies, game over)
- Structure générale d'un projet Unity
